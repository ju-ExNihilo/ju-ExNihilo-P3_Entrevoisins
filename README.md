	« Entrevoisins », est une application qui permet à des personnes d’un même quartier de se rendre des petits services : 
garde d’animaux, petit bricolage, troc d’objets, cours particuliers, de nombreuses options s’offrent aux utilisateurs ! 

	Fonctionnalités présente : 		
					* Liste des utilisateurs
					* Liste des favoris
					* Profil utilisateurs
					* Ajout d’un utilisateur
					* Ajout d’un utilisateur aux favoris
					* Suppression d’un utilisateur
					* Suppression d’un utilisateur des favoris

	Test présent : 

		- Test unitaire :
					* récupération de la liste d'utilisateurs
					* suppression d'un utilisateur
					* création d'un utilisateur
					* ajout d'un utilisateur aux favoris

		- Test instrumentalisé:
					* liste d'utilisateurs remplis
					* bouton supprimer 
					* lancement activité profil
					* profil initialisé avec les bonnes données
					* l'onglet favoris ne contient que les voisins marqués comme favoris
					* bouton supprimer de l'onglet favoris

	
	L’application n’étant pas encore déployé pour exécuter et compiler, Android Studio ( ou autre IDE )
 et un émulateur Android sont requis.
